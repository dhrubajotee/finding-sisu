Photos for individual blog posts go here.

For a post's cover photo (shown on cards), set in data/posts.js:
  cover: "/images/posts/the-silence.jpg"

For a post's gallery (shown next to the text on the post page), add to the
post's `gallery` array in data/posts.js, e.g.:
  gallery: [
    { src: "/images/posts/the-silence-1.jpg", orientation: "portrait", alt: "..." },
    { src: "/images/posts/the-silence-2.jpg", orientation: "landscape", alt: "..." },
  ]

Until a photo is added, the site shows a soft pastel placeholder automatically.
